# DnD Chess - 自走棋对战系统

基于 DnD 5E 属性体系 + 炉石酒馆战棋战斗机制的 C++ 自走棋游戏。

## 编译运行

```bash
g++ -std=c++17 -Isrc src/chess/ChessPiece.cpp src/player/Player.cpp src/player/AIPlayer.cpp src/battlefield/Battlefield.cpp src/game/Game.cpp src/main.cpp -o autochess
./autochess
```

## 游戏目标

你与 AI 对手各拥有 50 点生命，通过购买棋子、升级、合并进阶职业、排列阵型，在自动战斗中击败对手。一方生命归零即游戏结束。

## 游戏流程

每个回合分为三个阶段：

1. **准备阶段**：发金币 → AI 自动决策 → 玩家通过命令操作
2. **战斗阶段**：双方棋子自动对战（酒馆战棋式）
3. **结算阶段**：败方扣血（= 胜方存活棋子等级之和），胜方下回合 +1g

## 经济系统

- 每回合收入 = min(3 + 回合数 - 1, 10)，第 8 回合封顶 10g
- 战斗胜利下回合额外 +1g
- 初始 3 金币

准备阶段在 `> ` 提示符处输入命令：

### 商店操作

| 命令 | 说明 |

| `shop` | 查看当前商店 |
| `buy <n>` | 购买商店第 n 个棋子 |
| `refresh` | 刷新商店（花费 1g） |
| `us` | 升级商店（花费 = 当前商店等级 + 4） |

### 棋子管理

| 命令 | 说明 |
|---|---|
| `bench` | 查看备战席 |
| `board` | 查看战场阵型 |
| `place <b> <p>` | 将备战席第 b 个棋子放到战场位置 p（0-6） |
| `remove <p>` | 将战场位置 p 的棋子收回备战席 |
| `up <n>` | 升级备战席第 n 个棋子（花费 = 棋子金币 × 当前等级） |
| `merge <a> <b>` | 合并备战席第 a 和第 b 个棋子 |

### 其他

| 命令 | 说明 |

| `status` | 查看玩家完整状态 |

## 棋子系统

### 六维属性（DnD 5E）

STR 力量 / DEX 敏捷 / CON 体质 / INT 智力 / WIS 感知 / CHA 魅力

派生属性：
- HP = 10 + CON × 2
- AC = 10 + DEX 调整值 + 护甲加值
- 属性调整值 = (属性 - 10) / 2

### 7 个基础职业

| 职业 | 系别 | 主属性 | 技能 | 特殊状态 |
|---|---|---|---|---|
| 战士 Fighter | 近战 | STR | 二次风（30% 再攻击一次） | - |
| 野蛮人 Barbarian | 近战 | STR | 狂暴（HP<30% 伤害翻倍） | 嘲讽 |
| 圣武士 Paladin | 近战 | STR | 固定减伤 3 点 | 嘲讽 |
| 游侠 Ranger | 远程 | DEX | 猎人印记 | 潜行 |
| 吟游诗人 Bard | 远程 | CHA | 鼓舞（友军攻击力 +2） | - |
| 法师 Wizard | 法术 | INT | 火球术（溅射相邻 50%） | - |
| 牧师 Cleric | 法术 | WIS | 治疗术（治疗最低血友军） | - |

### 5 个进阶职业（多重继承）

两个**相同等级、相同系别、不同职业**的棋子可合并为进阶职业：

| 进阶职业 | 合并自 | 效果 |
|---|---|---|
| 狂战士 Berserker | 野蛮人 + 战士 | 狂暴 + 二次风 |
| 神卫士 Defender | 圣武士 + 野蛮人 | 减伤 + 狂暴 |
| 圣骑士 PaladinKnight | 战士 + 圣武士 | 二次风 + 减伤 |
| 魔射手 ArcaneArcher | 游侠 + 吟游诗人 | 射击 + 鼓舞 |
| 神谕者 Oracle | 法师 + 牧师 | 火球术 + 治疗术 |

### 升级

棋子等级 1-5 级，每升 1 级：主属性 +2、CON +1（HP 增长）。花费 = 棋子金币 × 当前等级。

## 战斗系统

### 攻击顺序
双方棋子按站位从左到右依次行动。

### 攻击目标选择
1. **潜行免疫**：潜行状态的敌方棋子不会被选为目标
2. **嘲讽优先**：若对方有嘲讽棋子，必须优先攻击
3. **随机攻击**：无嘲讽时随机选一个目标

### 攻击判定（DnD d20 命中检定）
1. 掷 d20 + 主属性调整值 + 武器加值 vs 目标 AC
2. 自然 1 → 大失败（下回合无法行动）
3. 自然 20 → 暴击（伤害 ×2）
4. 命中值 ≥ AC → 命中；< AC → 未命中

### 特殊状态
- **嘲讽**：必须被优先攻击
- **潜行**：不会被选为目标，主动攻击后破潜
- **大失败**：下回合无法行动

### 战后恢复
战斗结束后，所有存活棋子自动回满血。

## 阵型策略

- **嘲讽放左侧**（位置 0-2）：优先承受攻击，保护后排
- **输出放中间**（位置 3-4）：平衡出手顺序和存活
- **潜行放右侧**（位置 5-6）：避免被集火，等待输出时机

## 商店系统

- 商店等级 1-5，升级花费 = 当前等级 + 4
- 展示数量 = 商店等级 + 2
- 只刷新基础职业（进阶职业只能合并获得）
- 棋子等级 ≤ 商店等级才会出现
- 等级越高的棋子出现概率越低
